# Typing Kid VN

A fun and interactive web application designed to help children (and beginners) learn touch typing in Vietnamese.

## Features

- **Multiple Practice Modes**:
  - **Basic**: Learn keyboard rows (Home, Top, Bottom).
  - **Telex**: Practice Vietnamese tone marks using Telex.
  - **VNI**: Practice Vietnamese tone marks using VNI.
  - **Custom**: Input your own text to practice.
- **Visual Guides**: Interactive keyboard and hand placement indicators.
- **Progress Tracking**: Real-time WPM (Words Per Minute) and accuracy calculation.
- **Modern UI**: Dark mode with neon aesthetics and glassmorphism design.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: CSS (Modules/Variables), Framer Motion
- **Icons**: Lucide React

## Getting Started

1. Clone the repository:
   ```bash
   git clone git@github.com:minhtn012/typing-kid-vn.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
